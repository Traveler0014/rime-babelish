# 小鶴雙拼火星文

配方： ℞ **Babelish**

[Rime](https://rime.im) 雙拼火星文輸入方案

支持雙拼方案：

  - 小鶴雙拼

支持快捷輸入火星文、日期、農曆、計算結果等。

## 安裝

本方案依賴於

  - [朙月拼音](https://github.com/rime/rime-luna-pinyin) ℞ **`luna-pinyin`**

[東風破](https://github.com/rime/plum) 安裝口令： `bash rime-install double-pinyin`
